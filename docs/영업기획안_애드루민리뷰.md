## 데모 체험 방법

### 방법 1: 로그인 페이지에서 체험
1. https://adlumin-review.vercel.app/login 접속
2. 하단 "광고주 체험하기" 또는 "인플루언서 체험하기" 클릭
3. 바로 대시보드로 이동

### 방법 2: 메인 페이지에서 체험
1. https://adlumin-review.vercel.app 접속
2. "광고주로 체험하기" 또는 "인플루언서로 체험하기" 클릭

- "나가기" 버튼으로 데모 모드 종료 가능
- 데모 모드에서는 데이터 생성/수정이 차단되며, "샘플 모드에서는 사용할 수 없습니다" 알림이 표시됩니다

## A. 광고주 대상 영업

### 타겟
- 스마트스토어/쿠팡 셀러 (온라인 판매자)
- 로컬 매장 사장님 (카페, 맛집, 뷰티샵, 숙박업 등)
- 기업 마케팅 담당자

### 핵심 메시지
> "캠페인 하나 등록하면, 검증된 인플루언서가 알아서 신청합니다. 선정만 하세요."

### 시연 시나리오 (5분)

**1단계: 접속 및 로그인** (30초)
- 데모 URL 공유 → "광고주 체험하기" 클릭
- "회원가입 없이 바로 체험할 수 있습니다"

**2단계: 대시보드 둘러보기** (1분)
- 대시보드에서 전체 캠페인 현황 확인
- "지금 10개 캠페인이 등록되어 있고, 5개가 모집 중입니다"
- "삼성 갤럭시 버즈3 캠페인을 보시면 15명 중 12명이 이미 신청했습니다"

**3단계: 캠페인 등록 과정 설명** (2분)
- 캠페인 관리 > "새 캠페인" 클릭 → 등록 폼 보여주기
- "제품형은 배송으로, 방문형은 매장 방문으로 진행됩니다"
- 제품형 선택 → 16가지 채널 자동 노출 시연
- "구매평, 블로그, 인스타, 릴스, 클립, 틱톡, 쇼츠까지 전부 지원합니다"
- 카테고리 선택 → "식품, 뷰티, IT, 패션 등 10가지 카테고리"
- 방문형 전환 시연 → 지역 필터 노출
- 일정 설정 → "발표일과 마감일은 주말 선택이 안 됩니다. 직원분이 안내해야 하니까요"
- "제공내역, 검색키워드, 미션만 입력하면 등록 완료"

**4단계: 지원자 관리 시연** (1분)
- 캠페인 목록에서 "삼성 갤럭시 버즈3" 클릭
- 지원자 탭 확인
- "각 지원자의 SNS 채널, 팔로워 수, 매칭 점수를 볼 수 있습니다"
- "뷰티블로거 하은 님은 네이버 블로그 팔로워 1만 2천명, 매칭 점수 92점"
- "선정/미선정 버튼 하나로 처리. 선정되면 인플루언서에게 자동 알림이 갑니다"

**5단계: 클로징**
- "등록부터 선정, 콘텐츠 관리까지 원스톱. 레뷰, 리뷰플레이스와 동일한 기능인데, 구매평 체험단은 저희만 지원합니다."
- 프로필 페이지 → 회사 정보 확인 가능
- "궁금하신 점 있으시면 바로 가입하셔서 시작해보세요"

### 광고주 어필 포인트

| 어필 포인트 | 설명 | 데모에서 확인 가능 |
|-----------|------|----------------|
| 5분 등록 | 14개 항목만 입력하면 캠페인 개설 완료 | 캠페인 생성 폼 |
| 16가지 채널 | 구매평~쇼츠까지 전 채널 커버 | 채널 선택 그리드 |
| 자동 매칭 | 매칭 점수로 최적 인플루언서 추천 | 지원자 목록의 점수 |
| 인플루언서 검증 | 팔로워, 활동 이력 확인 가능 | 지원자 SNS 정보 |
| 제품형+방문형 | 셀러/플레이스 모두 지원 | 유형 선택 화면 |
| 주말 제외 일정 | 발표일/마감일 평일 한정 | 날짜 선택 시 |
| 콘텐츠 검수 | 게시 전 원고 검토 (개발 예정) | 향후 업데이트 |

### 예상 Q&A
**Q: 인플루언서는 어떻게 모집하나요?**
A: 플랫폼에 등록된 인플루언서가 캠페인을 보고 직접 신청합니다. 저희가 별도로 인플루언서 모집과 교육도 진행합니다.

**Q: 비용은 어떻게 되나요?**
A: 캠페인 건 당 과금이며, 월 구독료는 없습니다. 상세 가격은 별도 안내드리겠습니다.

**Q: 레뷰, 리뷰플레이스와 뭐가 다른가요?**
A: 같은 체험단 기능에 더해, 구매평 체험단 8종과 검색유입 업그레이드를 저희만 지원합니다. 셀러 맞춤 기능에 특화되어 있습니다.

**Q: 구매평 체험단이 뭔가요?**
A: 인플루언서가 실제로 제품을 구매한 후 스마트스토어에 구매평을 작성하는 서비스입니다. 블로그, 인스타 등과 결합도 가능합니다.

## B. 인플루언서 대상 영업

### 타겟
- 네이버 블로거 (일 방문자 100명 이상)
- 인스타그래머 (팔로워 500명 이상)
- 유튜버/틱토커 (구독자 500명 이상)
- 리뷰 활동에 관심 있는 일반인

### 핵심 메시지
> "맛집도 가고, 제품도 받고, 내 SNS로 수익까지. 무료로 시작하세요."

### 시연 시나리오 (3분)

**1단계: 접속** (20초)
- "인플루언서 체험하기" 클릭
- "무료 가입만 하면 바로 시작할 수 있어요"

**2단계: 캠페인 탐색** (1분 30초)
- 캠페인 탐색 클릭
- "지금 이런 캠페인들이 모집 중입니다"
- 데모 캠페인 보여주기:
  - "삼성 갤럭시 버즈3를 무료로 받고 구매평만 쓰면 됩니다"
  - "성수동 카페에 가서 블로그 리뷰만 올리면 돼요. 2인 브런치가 무료"
  - "올리브영 스킨케어 세트를 받고 인스타에 올리면 끝"
- 필터 시연: "제품형/방문형, 블로그/인스타/틱톡 등 원하는 것만 골라서"
- 캠페인 상세 클릭 → "제공내역, 미션, 일정이 다 나와있어요. 신청하기 한 번이면 끝"

**3단계: 내 신청 현황** (1분)
- 대시보드 또는 "내 신청" 클릭
- "신청한 캠페인 상태가 바로 보입니다"
- "선정되면 제품을 받거나 매장에 방문해서 리뷰만 쓰면 돼요"
- "기존 7건 완료, 평점 4.6점 — 활동하면 등급도 올라갑니다"

**4단계: 클로징**
- "블로그 하나만 있어도 됩니다. 무료이고, 지금 바로 가입해서 시작해보세요"

### 인플루언서 어필 포인트

| 어필 포인트 | 설명 | 데모에서 확인 가능 |
|-----------|------|----------------|
| 무료 가입 | 비용 없이 바로 시작 | 회원가입 페이지 |
| 제품 무료 제공 | 갤럭시 버즈, 스킨케어 등 | 캠페인 상세 |
| 맛집/카페 무료 | 성수동 카페, 제주 호텔 등 | 방문형 캠페인 |
| 간편 신청 | 한 번 클릭으로 끝 | 신청 버튼 |
| 등급 시스템 | 실버→골드→프리미어 | 프로필 페이지 |
| 다양한 채널 | 블로그, 인스타, 틱톡, 쇼츠 | 캠페인 필터 |

### 예상 Q&A
**Q: 팔로워가 적어도 되나요?**
A: 네. 팔로워 수보다 콘텐츠 품질이 중요합니다. 블로그 일 방문자 100명부터 신청 가능해요.

**Q: 어떤 SNS가 필요해요?**
A: 네이버 블로그, 인스타그램, 유튜브, 틱톡 중 하나만 있으면 됩니다.

**Q: 돈을 내야 하나요?**
A: 아니요. 인플루언서는 완전 무료입니다. 오히려 제품을 받고 포인트도 적립됩니다.

**Q: 선정은 어떻게 되나요?**
A: 광고주가 신청자 중에서 선정합니다. 채널 적합성, 콘텐츠 품질, 과거 활동 이력을 봅니다.

```
안녕하세요, 애드루민입니다.

체험단 마케팅 플랫폼을 소개드립니다.
구매평, 블로그, 인스타, 틱톡 등 16가지 채널을 지원하며,
```

**인플루언서용:**
```
안녕하세요!

무료로 제품 받고 리뷰 쓰는 체험단 플랫폼을 소개해드려요.
갤럭시 버즈, 스킨케어, 맛집, 호텔 등 다양한 캠페인이 있습니다.

체험해보기: https://adlumin-review.vercel.app/login
→ "인플루언서 체험하기" 클릭

블로그나 인스타 계정만 있으면 무료로 시작할 수 있어요!
```

### 데모 시연 팁
1. **광고주에게는** 캠페인 등록 폼과 지원자 관리를 중심으로 설명
2. **인플루언서에게는** 캠페인 탐색과 제공내역(무료 제품/서비스)을 중심으로 설명
3. **역할 전환** — 오른쪽 하단 위젯에서 광고주↔인플루언서 즉시 전환 가능. "광고주가 올린 캠페인이 인플루언서에게는 이렇게 보입니다" 식으로 양쪽 시점 비교 시연
4. **데이터가 현실적** — 삼성, 올리브영, 성수동 카페 등 실제 있을 법한 캠페인으로 구성

5. **카카오/네이버 로그인** — 원클릭 간편 가입
6. **모바일 앱** — iOS/Android 전용 앱
